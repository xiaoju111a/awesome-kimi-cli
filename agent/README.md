# Agent Examples

A collection of example agents built with Kimi Agent SDK.

## Available Examples

### [hello-agent](./hello-agent)

A simple example demonstrating basic Kimi Agent SDK usage.

**Features:**
- Basic session creation and management
- Streaming response handling
- Multi-turn conversation support
- Tool call visibility
- Token usage statistics
- Error handling

**Quick Start:**
```bash
cd hello-agent
npm install
cp .env.example .env
# Edit .env and add your KIMI_API_KEY
npm start
```

## Prerequisites

All examples require:
- Node.js 18+
- [Kimi CLI](https://github.com/MoonshotAI/kimi-cli) installed
- Moonshot AI API Key ([Get one here](https://platform.moonshot.ai/))

## Getting Started

1. Choose an example project
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Configure environment variables (copy `.env.example` to `.env`)
5. Run the example: `npm start`
