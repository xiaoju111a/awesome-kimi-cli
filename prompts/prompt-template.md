# Prompt Template System

## Best Practice Template

```
## 上下文设置
你是一位[具体角色]，拥有[相关经验]...

## 任务定义
需要完成[具体功能]，服务于[目标用户]...

## 技术要求
- 技术栈：[具体技术]
- 性能要求：[性能指标]
- 安全要求：[安全标准]

## 输出格式
请提供：
1. 完整的代码实现
2. 详细的注释说明
3. 单元测试用例
4. 使用示例
```

## Iterative Optimization Flow

1. **初始Prompt**：清晰定义需求和约束
2. **代码生成**：AI生成初步代码
3. **代码审查**：人工审查和测试
4. **反馈优化**：基于问题改进prompt
5. **最终验证**：确保代码质量和性能
