# Fullstack Development Prompts

## Fullstack Application

```
创建一个完整的全栈应用：任务管理系统

前端要求：
- React + TypeScript + Tailwind CSS
- 支持任务CRUD操作
- 响应式设计，支持移动端
- 使用Redux Toolkit进行状态管理

后端要求：
- Node.js + Express + TypeScript
- MongoDB数据库
- RESTful API设计
- JWT认证
- 输入验证和错误处理

请提供完整的项目结构、核心代码和部署配置。
```

## Security Best Practices

```
实现一个安全的Web应用，包含：

前端安全：
- CSP策略配置
- XSS防护
- CSRF防护
- 输入验证和净化

后端安全：
- SQL注入防护
- 认证和授权
- API安全
- 敏感数据加密

请提供完整的代码实现和安全配置。
```
