# Frontend Development Prompts

## UI/UX Design

```
作为资深前端工程师，请基于以下需求设计React组件：
- 需求：现代化的电商产品卡片
- 要求：响应式设计、支持暗色模式、包含悬停效果
- 技术栈：React + TypeScript + Tailwind CSS
- 性能：需要支持虚拟滚动

请提供完整的组件代码，包括Props接口定义。
```

## Performance Optimization

```
分析以下React代码的性能瓶颈，并提供优化方案：
[代码片段]

要求：
1. 识别不必要的重渲染
2. 建议使用React.memo、useMemo、useCallback
3. 提供优化后的代码
4. 解释优化原理
```

## Component Development

```
创建一个可复用的表单组件，包含以下功能：
- 支持多种输入类型（文本、数字、选择框等）
- 内置表单验证
- 支持自定义验证规则
- 提供TypeScript类型定义
- 包含单元测试

请遵循React最佳实践，使用现代Hook模式。
```

```
优化以下React应用的性能：
- 减少首屏加载时间

- 添加缓存策略
- 支持服务端渲染(SSR)

请提供具体的优化方案和实现代码。
```
