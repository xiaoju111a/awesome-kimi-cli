# Contributing Guide

Thanks for your interest in Awesome Kimi CLI! We welcome all contributions.

## How to Contribute

### Adding New Resources

1. Fork this repository
2. Create your branch: `git checkout -b add-new-resource`
3. Add your resource to README.md
4. Commit your changes: `git commit -m 'Add: resource name'`
5. Push to the branch: `git push origin add-new-resource`
6. Submit a Pull Request

### Format

Use the following format when adding resources:

```markdown

```

### Commit Message Convention

- `Add: xxx` - Add new resource
- `Update: xxx` - Update existing resource
- `Remove: xxx` - Remove invalid resource
- `Fix: xxx` - Fix links or formatting issues

## Code of Conduct

- Be friendly and respectful
- Provide constructive feedback
- Focus on the best interests of the community

## Issues

If you find any problems, feel free to [open an issue](https://github.com/MoonshotAI/kimi-cli/issues).
